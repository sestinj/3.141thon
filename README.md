# 3.141thon
is a powerful, concise scripting language similar to Python (after all, that's what it is built on). Currently, 3.141thon supports the following features (but it is still progressing):

- Expressions
- Variables
- Functions
- Nested scopes
- Arrays
- For loops
- Dictionaries
- Subscripting
